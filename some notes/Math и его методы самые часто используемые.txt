Math.abs(x) - возвращает абсолютное значение числа x.
Math.ceil(x) - возвращает наименьшее целое число, большее или равное аргументу x.
Math.floor(x) - возвращает наибольшее целое число, меньшее или равное аргументу x.
Math.round(x) - возвращает значение числа x, округленное до ближайшего целого.
Math.max(x1, x2, ..., xn) - возвращает наибольшее из заданных чисел.
Math.min(x1, x2, ..., xn) - возвращает наименьшее из заданных чисел.
Math.pow(x, y) - возвращает значение x в степени y (x^y).
Math.sqrt(x) - возвращает квадратный корень из числа x.
Math.random() - возвращает случайное число между 0 и 1.
Math.sin(x) - возвращает синус угла, заданного в радианах.
Math.cos(x) - возвращает косинус угла, заданного в радианах.
Math.tan(x) - возвращает тангенс угла, заданного в радианах.
